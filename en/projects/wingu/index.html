wingu

Multiplatform framework to implement location features into Apps and Web

As part of the wingu core team we developed over the last 3 years an extensive platform and accompanied SDK's that enables developers to quickly and easily implement location features into mobile Apps and Web. The API service platform supports extensive proximity and data management and works in close connection with the advanced native (Swift / Kotlin) mobile SDKs. The wingu platform enables developers to create proximity triggers and location based content and interaction by only including the wingu SDK in their mobile app.

During the project we delivered

  • Proximity trigger management solution - REST API build on Symfony.
  • Native SDKs (for iOS, Android and PHP)
  • Native apps for several clients
  • Extensive app generator for white label
  • Secure and comprehensive API for content creation and delivery
  • Full CMS to create rich content and interactions
  • Data analytics and reporting service in compliance with GDPR
  • All DevOps services based on Heroku & Amazon AWS

Quick overview of used technologies

  • Swift & Kotlin
  • PHP, Symfony framework
  • AngularJS & Typescript
  • Heroku, AWS, RabbitMQ, MongoDB, PostgreSQL Redis
