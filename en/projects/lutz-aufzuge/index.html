Lutz Aufzüge

A REST API for Smart Elevators

Elevators are designed for the simple act of traveling between floors. Hardly anyone is aware of the complex technology behind the scenes. Smart technology that makes elevators secure, faster, easier and more energy efficient than older elevators.

We created together with and for the innovative elevator company Lütz Aufzüge (made in Germany) a comprehensive API that will enable a whole range of new and innovative technically possibilities ranging from predictive maintenance, issue alerting to smart elevators.

Our API developers worked close together with the development team from Lutz Aufzüge, and created a basic REST api architecture for future development.
